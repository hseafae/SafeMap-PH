# SafeMap PH

SafeMap PH is a web-based system for anonymous community incident reporting
and safety hotspot visualization designed for Philippine LGUs and PNP workflows.

## Features
- Anonymous incident reporting
- Interactive map with heat and verified layers
- Admin validation workflow
- Get Help page with verified Philippine emergency hotlines
- AI chatbot for user assistance

## Tech Stack
- HTML, CSS (Bootstrap 5)
- JavaScript (ES Modules)
- Leaflet.js for interactive maps

## Disclaimer
SafeMap PH does not replace official police reporting.
For emergencies, please call **911**.
